# 🤖 30 Algoritmos de IA — Aprenda do Zero

Um projeto educacional e interativo focado em ensinar 30 dos principais conceitos e algoritmos de Inteligência Artificial e Machine Learning de forma simples, visual e gamificada.

## 📖 Sobre o Projeto

O **30 Algoritmos de IA** foi criado para desmistificar o mundo do aprendizado de máquina. Através de uma interface moderna e responsiva, o usuário pode explorar diferentes categorias de algoritmos, aprender de forma teórica e prática, e testar seus conhecimentos de maneira divertida.

As fichas de estudo para cada algoritmo incluem:
- **Explicação rápida e direta:** O que é e como funciona matematicamente/teoricamente.
- **Analogias reais:** Exemplos fáceis do dia a dia para facilitar o entendimento de iniciantes.
- **Casos de uso:** Onde vemos essa tecnologia no mundo real.
- **Prós e Contras:** Principais forças e limitações.
- **Exemplo de Código (Python):** Snippets com bibliotecas populares (Scikit-learn, PyTorch, XGBoost).
- **Quiz de fixação:** Uma pergunta interativa para confirmar o que você acabou de aprender!

## ✨ Funcionalidades em Destaque

- 💾 **Progresso Local:** O app utiliza o `localStorage` do seu navegador para salvar se você completou ou não os quizzes. Acompanhe a sua barra de progresso encher (0 a 30) ao longo dos estudos!
- 🔍 **Filtro Inteligente:** Filtre algoritmos por nome ou transite facilmente pelas abas de categorias.
- 📱 **Totalmente Responsivo:** Layout fluido que se recalcula e ajusta automaticamente em celulares, tablets ou telas super largas (ultrawide).
- 🎨 **UI/UX Polida:** Um _Dark Mode_ lindíssimo com design contemporâneo, micro-interações, efeitos de foco, blur e transições de tela animadas.
- ⚡ **Extremamente Leve:** Zero dependências (sem bibliotecas externas) na construção da estrutura web (Single Page App feito 100% em Vanilla).

## 🗂️ Categorias Abordadas

Trazemos da estatística clássica às arquiteturas modernas adotadas pelo ChatGPT:

1. **Supervisionado:** Regressão (Linear/Logística), Árvores, SVM, KNN, Naive Bayes.
2. **Ensemble:** Random Forest, Gradient/Ada-Boosting, XGBoost.
3. **Clustering:** k-Means (+k-Means++), Hierarchical, DBSCAN.
4. **Redução Dimensional:** PCA, t-SNE.
5. **Aprendizado por Reforço (RL):** Q-Learning, SARSA, DQN, Policy Gradient, Actor-Critic, MDP.
6. **Deep Learning:** ANNs clássicas, CNN (Visão), RNN e LSTM (Série temporal/Texto), Transformers (Modelos de base para LLMs) e Autoencoders.
7. **Anomalia/Outros:** Algoritmos Genéticos e Isolation Forest.

## 🚀 Como executar o projeto

Isso é o melhor de tudo: o projeto inteiro roda diretamente no seu computador sem necessidade de instalação de dependências ou uso de terminais.

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/algorithms_quiz.git
   ```
2. Entre na pasta recém criada:
   ```bash
   cd algorithms_quiz
   ```
3. Dê dois cliques no arquivo `index.html` para abrir diretamente no seu navegador.
   - *Dica opcional:* Se usar VS Code, você pode usar a extensão **Live Server** para obter recarregamento dinâmico.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Marcadores semânticos.
- **CSS3:** Variáveis de design (`Custom Properties`), CSS Grid e Flexbox flexíveis, animações de estado.
- **JavaScript (ES6 Vanilla):** Interatividade do front-end, renderização limpa do DOM baseada em matrizes, manipulação do LocalStorage e estado dos quizzes.
